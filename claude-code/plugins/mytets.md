→ [[claude-code/README|← Назад к обзору]]

## Описание

Локальный плагин. Активирует украинский театральный абсурдный стиль общения с гиперболическим пафосом и эпической драмой.

## Установка

Локальный плагин — не из маркетплейса. Восстановить из бэкапа:

```bash
# Скопировать папку mytets в:
~/.claude/plugins/marketplaces/local-desktop-app-uploads/mytets/
```
