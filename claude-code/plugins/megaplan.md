## Описание

Локальный плагин для интеграции с Megaplan CRM.

## Установка

Локальный плагин — не из маркетплейса. Восстановить из бэкапа:

```bash
# Скопировать папку megaplan в:
~/.claude/plugins/marketplaces/local-desktop-app-uploads/megaplan/
```
