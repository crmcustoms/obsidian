# Planfix AI Tools & Examples

Дополнения к основному SKILL.md на основе анализа лучших практик (2024-2025) и репозитория viasite/planfix-tools.

## 🛠 Полезные наработки (viasite/planfix-tools)

Репозиторий ценен примерами прямой работы с API на Node.js без лишних оберток.

### Ключевые файлы для изучения:
- `src/api.js` — минималистичный клиент для запросов.
- `src/commands/` — примеры реализации логики (обновление цен, контактов).

### Чему поучиться у viasite:
1. **Прозрачная авторизация** — автоматический проброс токенов из конфига.
2. **XML -> JSON** — Планфикс отдает XML, ребята используют `xml2js` для удобной работы в JS/TS.
3. **Кеширование** — использование `lowdb` для кратковременного хранения цен, чтобы не спамить API.

## 💡 Лучшие практики (2024-2025)

### 1. Переменные в сценариях (Must have)
Используй сложные шаблоны для автоматических ответов:
- `{{Задача.Контрагент.Имя}}` — классика.
- Новинка: использование переменных в заголовках HTTP-запросов сценариев.

### 2. Работа с ИИ (Planfix AI)
- **Суммаризация:** Автоматически сжимать длинные обсуждения в задаче через Planfix AI.
- **Анализ эмоций:** Интегрируй сценарии для детекта негатива в комментариях (настройка через события "Добавлен новый комментарий").

### 3. Веб-хуки и REST API
- **Обновление в v18:** В HTTP-запросах теперь можно передавать файлы.
- **Событие SLA:** "Превышено время нахождения задачи в статусе" — критично для наших клиентов на техподдержке.

### 4. Интеграция с Telegram (Official Bot Mode)
- Настраивать ботов в группах через API так, чтобы они видели историю правок сообщений (`message_id`).
