# Секреты и безопасность
В этом репозитории ЗАПРЕЩЕНО хранить:
1. API токены (Notion, Planfix, Google).
2. Пароли и ключи SSH.
3. Личные номера телефонов и секретные ссылки.

Все ключи хранятся ТОЛЬКО локально в `/home/node/.config/` и прописываются в системные переменные.
При обнаружении секрета в Markdown-файле — немедленно удалить и заменить на заглушку `[SECRET_HIDDEN]`.
