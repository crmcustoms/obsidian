## Товары (Offer)

### Эндпоинты

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/offer` | Список товаров |
| GET | `/api/v3/offer/{id}` | Получить товар |
| POST | `/api/v3/offer` | Создать товар |
| POST | `/api/v3/offer/{id}` | Обновить товар |
| DELETE | `/api/v3/offer/{id}` | Удалить товар |
| GET | `/api/v3/offer/{id}/balances` | Остатки на складах |
| GET | `/api/v3/offer/{id}/parts` | Комплектующие |
| GET | `/api/v3/offerCategory` | Категории товаров |
| GET | `/api/v3/offerCategory/{id}` | Получить категорию |
| GET | `/api/v3/offerCategory/{id}/fields` | Поля категории |
| POST | `/api/v3/offerPart` | Добавить комплектующее |
| DELETE | `/api/v3/offerPart/{id}` | Удалить комплектующее |

## Финансовые операции (FinOperation)

| Метод | URL | Описание |
|-------|-----|---------|
| POST | `/api/v3/finOperation` | Создать операцию |
| GET | `/api/v3/finOperation/{id}` | Получить операцию |
| POST | `/api/v3/finOperation/{id}` | Обновить операцию |
| POST | `/api/v3/finOperation/createTransfer` | Перевод между счетами |
| GET | `/api/v3/finOperation/types` | Типы операций |

### Поля FinOperation

```json
{
  "contentType": "FinOperation",
  "id": "string",
  "sum": "Money",
  "date": "DateTime",
  "type": "FinOperationType",
  "comment": "string",
  "deal": "Deal",
  "invoice": "Invoice",
  "contractor": "ContractorCompany|ContractorHuman",
  "employee": "Employee"
}
```

## Накладные (Consignment)

Создаются из сделок при отгрузке товаров.

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/consignment` | Список накладных |
| GET | `/api/v3/consignment/{id}` | Получить накладную |
