# Пагинация и управление полями — Megaplan API v3

← [[Megaplan API — Index]]

---

## Тип пагинации

Все списки используют **cursor-based** пагинацию, не offset.

---

## Параметры

| Параметр | Тип | Описание |
|----------|-----|---------|
| `limit` | integer | Элементов на странице (обычно макс. 100) |
| `pageAfter` | BaseEntity | Следующая страница **после** этой сущности |
| `pageBefore` | BaseEntity | Страница строго **до** этой сущности |
| `pageWith` | BaseEntity | Страница, содержащая эту сущность |
| `fields[]` | any | Дополнительные поля для включения в ответ |
| `onlyRequestedFields` | boolean | Вернуть **только** перечисленные поля |
| `sortBy[]` | SortField[] | Сортировка |

---

## Пример итерации всех записей

```python
import httpx, json

client = httpx.Client(
    base_url="https://account.megaplan.ru",
    headers={"Authorization": "Bearer TOKEN"}
)

page_after = None
while True:
    params = {"limit": 100}
    if page_after:
        params["pageAfter"] = json.dumps({"contentType": "Deal", "id": page_after})

    r = client.get("/api/v3/deal", params=params)
    items = r.json()["data"]

    if not items:
        break

    process(items)
    page_after = items[-1]["id"]
```

## Управление полями

### Запросить дополнительные поля
```
GET /api/v3/deal/{id}?fields[]=invoices&fields[]=positions&fields[]=possibleTransitions
```

### Только нужные поля (экономия трафика)
```
GET /api/v3/deal?fields[]=name&fields[]=price&fields[]=state&onlyRequestedFields=true
```

---

## Сортировка

```json
sortBy: [
  {"contentType": "SortField", "fieldName": "timeCreated", "desc": true}
]
```
