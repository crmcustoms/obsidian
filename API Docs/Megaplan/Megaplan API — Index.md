## Общие правила API

- Все обновления через `POST /{entity}/{id}` (не PUT/PATCH)
- Ссылка на сущность: `{"contentType": "Deal", "id": "123"}`
- ID всегда `string`, даже если выглядит как число
- `isDropped: true` — мягко удалённая сущность
- `possibleActions[]` — список разрешённых действий для текущего юзера
- Список возвращает меньше полей чем объект по id
