# Счета (Invoice) — Megaplan API v3

← [[Megaplan API — Index]]

---

## Эндпоинты

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/invoice` | Список счетов |
| GET | `/api/v3/invoice/{id}` | Получить счёт |
| POST | `/api/v3/invoice` | Создать счёт |
| POST | `/api/v3/invoice/{id}` | Обновить счёт |
| DELETE | `/api/v3/invoice/{id}` | Удалить счёт |
| GET | `/api/v3/invoice/{id}/rows` | Позиции счёта |
| GET | `/api/v3/invoice/realAccounts` | Расчётные счета |
| GET | `/api/v3/invoice/template` | Шаблоны счетов |
| GET | `/api/v3/invoice/templateVariables` | Переменные шаблонов |

---

## Параметры GET /api/v3/invoice

| Параметр | Описание |
|----------|---------|
| `statusFilterId` | ID сохранённого InvoiceFilter |
| `periodFilterId` | ID фильтра по периоду |
| `limit` | Кол-во элементов |
| `pageAfter` | Cursor пагинация |

> ⚠️ **Нет поиска по `name` и `q`**. Подробнее: [[Фильтрация#Поиск Invoice по name]]

---

## Статусы

| Значение | Смысл |
|----------|-------|
| `created` | Создан |
| `drawn` | Выставлен |
| `paid` | Оплачен |
| `rejected` | Отклонён |

## Создание счёта

```json
POST /api/v3/invoice
{
  "contentType": "Invoice",
  "invoiceDate": {"contentType": "DateOnly", "year": 2024, "month": 3, "day": 15},
  "number": "1234",
  "name": "Счёт за услуги",
  "recipient": {"contentType": "Payer", "id": "payer_id"},
  "payer": {"contentType": "Payer", "id": "payer_id"},
  "consignee": {"contentType": "Payer", "id": "payer_id"},
  "rows": [
    {
      "contentType": "OfferRow",
      "name": "Услуга разработки",
      "price": {"contentType": "Money", "value": 15000, "currency": "RUB"},
      "count": 1
    }
  ]
}
```

## Поиск счёта по name (обходной путь)

### Вариант 1 — фильтр на клиенте (проще)
```python
invoices = get_all("/api/v3/invoice")
found = [i for i in invoices if i["name"] == "Счёт №123"]
```

### Вариант 2 — временный InvoiceFilter
```
POST /api/v3/invoiceFilter  →  GET /api/v3/invoice?statusFilterId={id}  →  DELETE /api/v3/invoiceFilter/{id}
```

Подробнее: [[Фильтрация#Временный фильтр для Invoice/Contractor]]
