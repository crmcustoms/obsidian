## Параметры GET /api/v3/deal

| Параметр | Описание |
|----------|---------|
| `q` | Поиск по названию / номеру |
| `filter` | TradeFilter — id или inline-конфиг |
| `status` | Фильтр по статусу ProgramState |
| `limit` | Кол-во элементов |
| `pageAfter` | Cursor пагинация |
| `fields[]` | Дополнительные поля |

## Создание сделки

> ⚠️ Поле `program` (схема/воронка) — **обязательно**.

```json
POST /api/v3/deal
{
  "contentType": "Deal",
  "name": "Новая сделка",
  "program": {"contentType": "Program", "id": "1"},
  "contractor": {"contentType": "ContractorCompany", "id": "42"},
  "manager": {"contentType": "Employee", "id": "5"},
  "price": {"contentType": "Money", "value": 50000, "currency": "RUB"}
}
```

---

## Смена статуса

```json
POST /api/v3/deal/{id}/applyTransition
{
  "contentType": "ProgramTransition",
  "id": "transition-id"
}
```

Список доступных переходов:
```
GET /api/v3/deal/{id}?fields[]=possibleTransitions
```

## Результат сделки

| Значение | Смысл |
|----------|-------|
| `active` | В работе |
| `positive` | Успешно завершена |
| `negative` | Провалена |
