# Сотрудники (Employee) — Megaplan API v3

← [[Megaplan API — Index]]

---

## Эндпоинты

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/employee` | Список сотрудников |
| POST | `/api/v3/employee/{id}` | Обновить сотрудника |
| DELETE | `/api/v3/employee/{id}` | Удалить сотрудника |
| POST | `/api/v3/employee/{id}/fire` | Уволить |
| POST | `/api/v3/employee/{id}/restoreFired` | Восстановить уволенного |
| POST | `/api/v3/employee/{id}/block` | Заблокировать |
| POST | `/api/v3/employee/{id}/unblock` | Разблокировать |
| GET | `/api/v3/employee/{id}/subordinates/{mode}` | Подчинённые |
| GET | `/api/v3/employee/{id}/delegatedEmployees` | Делегированные сотрудники |
| GET | `/api/v3/employee/{id}/vacations` | Отпуска |
| GET | `/api/v3/employee/{id}/closestVacation` | Ближайший отпуск |
| GET | `/api/v3/employee/{id}/activity` | Активность |
| GET | `/api/v3/employee/{id}/invisibleDepartments` | Скрытые отделы |
| GET | `/api/v3/employee/birthdays` | Дни рождения |
| GET | `/api/v3/employee/controlled` | Подконтрольные сотрудники |
| GET | `/api/v3/employee/inviteLink` | Ссылка-приглашение |
| GET | `/api/v3/employeeStatus` | Статусы сотрудников |
| GET | `/api/v3/department` | Список отделов |
| GET | `/api/v3/currentUser` | Текущий пользователь |

---

## Параметры GET /api/v3/employee

| Параметр | Описание |
|----------|---------|
| `q` | Поиск по имени / email |
| `filter` | EmployeeFilter — id или inline |
| `limit` | Кол-во элементов |
| `pageAfter` | Cursor пагинация |

## Поиск сотрудника по имени

```
GET /api/v3/employee?q=Иван+Иванов&limit=10
```

---

## Увольнение

```json
POST /api/v3/employee/{id}/fire
{
  "contentType": "Employee",
  "fireDay": {"contentType": "DateTime", "value": "2024-03-15T00:00:00+03:00"}
}
```

---

## Восстановление уволенного

```
POST /api/v3/employee/{id}/restoreFired
```

---

## Структура организации

```
GET /api/v3/department                    — все отделы
GET /api/v3/employee/{id}/subordinates/all — все подчинённые
GET /api/v3/employee/{id}/subordinates/direct — прямые подчинённые
```
