# Клиенты (Contractor) — Megaplan API v3

← [[Megaplan API — Index]]

Два типа: `ContractorCompany` (юрлицо/компания) и `ContractorHuman` (физлицо).

---

## Эндпоинты — общие

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/contractor` | Все клиенты (компании + физлица) |
| GET | `/api/v3/contractor/{id}` | Клиент по id |
| DELETE | `/api/v3/contractor/{id}` | Удалить |
| GET | `/api/v3/contractor/{id}/deals` | Сделки клиента |
| GET | `/api/v3/contractor/{id}/invoices` | Счета клиента |
| GET | `/api/v3/contractor/{id}/tasks` | Задачи клиента |
| GET | `/api/v3/contractor/{id}/payers` | Реквизиты |
| POST | `/api/v3/contractor/{id}/payers` | Добавить реквизит |
| POST | `/api/v3/contractor/{id}/merge` | Объединить дубликаты |
| GET | `/api/v3/contractor/{id}/possibleDuplicates` | Возможные дубли |
| GET | `/api/v3/contractor/types` | Типы клиентов |

## ContractorCompany — Компания

### Эндпоинты

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/contractorCompany` | Список компаний |
| GET | `/api/v3/contractorCompany/{id}` | Получить компанию |
| POST | `/api/v3/contractorCompany` | Создать компанию |
| POST | `/api/v3/contractorCompany/{id}` | Обновить компанию |
| DELETE | `/api/v3/contractorCompany/{id}` | Удалить компанию |
| GET | `/api/v3/contractorCompany/{id}/contacts` | Контактные лица (физлица) |
| GET | `/api/v3/contractorCompany/{id}/deals` | Сделки |
| GET | `/api/v3/contractorCompany/{id}/invoices` | Счета |
| POST | `/api/v3/contractorCompany/{id}/responsibles` | Назначить ответственного |
| GET | `/api/v3/contractorCompany/extraFields` | Доп. поля |
| GET | `/api/v3/contractorCompany/tags` | Теги |

### Схема полей

```json
{
  "contentType": "ContractorCompany",
  "id": "string",
  "name": "string",
  "position": "string",
  "type": "ContractorType",
  "status": "ContractorStatus",
  "responsibles": "(Employee|Group)[]",
  "contacts": "ContractorHuman[]",
  "contactInfo": "(ContactInfo|Address)[]",
  "description": "string",
  "birthday": "DateOnly",
  "preferTransport": "email|sms|skype|informer",
  "advertisingWay": "ContractorAdvertisingWay",
  "activityType": "ContractorActivityType",
  "payers": "Payer[]",
  "partners": "(ContractorCompany|ContractorHuman)[]",
  "competitors": "(ContractorCompany|ContractorHuman)[]",
  "isPublic": "boolean",
  "isDropped": "boolean",
  "isFavorite": "boolean",
  "timeCreated": "DateTime",
  "timeUpdated": "DateTime",
  "nextCommunicationDate": "DateTime|DateOnly",
  "lastDealDate": "DateTime",
  "countDeals": "integer",
  "countActiveDeals": "integer",
  "summDeals": "Money",
  "countInvoice": "integer",
  "summInvoice": "Money",
  "balance": "Money",
  "tags": "Tag[]"
}
```

### Создание компании

```json
POST /api/v3/contractorCompany
{
  "contentType": "ContractorCompany",
  "name": "ООО Ромашка",
  "contactInfo": [
    {
      "contentType": "ContactInfo",
      "type": {"contentType": "ContactInfoType", "id": "phone"},
      "value": "+7 999 123-45-67",
      "isMain": true
    },
    {
      "contentType": "ContactInfo",
      "type": {"contentType": "ContactInfoType", "id": "email"},
      "value": "info@romashka.ru"
    }
  ],
  "responsibles": [{"contentType": "Employee", "id": "5"}]
}
```

## ContractorHuman — Физлицо

### Эндпоинты

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/contractorHuman` | Список физлиц |
| GET | `/api/v3/contractorHuman/{id}` | Получить физлицо |
| POST | `/api/v3/contractorHuman` | Создать физлицо |
| POST | `/api/v3/contractorHuman/{id}` | Обновить физлицо |
| DELETE | `/api/v3/contractorHuman/{id}` | Удалить физлицо |
| GET | `/api/v3/contractorHuman/guests` | Клиенты с гостевым доступом |

### Дополнительные поля (к полям CompractorCompany)

```json
{
  "contentType": "ContractorHuman",
  "firstName": "string",
  "lastName": "string",
  "middleName": "string",
  "name": "string",
  "gender": "male|female",
  "company": "ContractorCompany",
  "uid": "integer",
  "loginEmail": "string",
  "canLogin": "boolean",
  "lastOnline": "DateTime"
}
```
