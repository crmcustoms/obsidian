## DateTime

```json
{
  "contentType": "DateTime",
  "value": "2024-03-15T10:30:00+03:00"
}
```

Формат: ISO 8601 с timezone. **Всегда указывать timezone.**

---

## DateOnly

```json
{
  "contentType": "DateOnly",
  "year": 2024,
  "month": 3,
  "day": 15
}
```

## Money

```json
{
  "contentType": "Money",
  "value": 10000.0,
  "currency": "RUB",
  "valueInMain": 10000.0,
  "rate": 1.0
}
```

## BaseEntity (ссылка на сущность)

При передаче ссылки на любую сущность достаточно двух полей:
```json
{"contentType": "Deal", "id": "123"}
```

Не нужно передавать все поля объекта — только `contentType` и `id`.

---

## ContactInfo

```json
{
  "contentType": "ContactInfo",
  "type": {"contentType": "ContactInfoType", "id": "phone"},
  "value": "+7 999 123-45-67",
  "isMain": true
}
```

Типы (`id`): `phone`, `email`, `skype`, `site`, `icq`, `jabber`, `twitter`, `facebook`

## Address

  "isMain": true
}

## Payer (реквизиты)

Используется в [[Счета|Invoice]] и [[Сделки|Deal]].

```json
{
  "contentType": "Payer",
  "id": "string",
  "name": "ООО Ромашка",
  "type": "string",
  "inn": "7701234567",
  "kpp": "770101001",
  "ogrn": "string",
  "bik": "044525225",
  "bank": "Сбербанк",
  "currentAccount": "40702810000000000001",
  "correspondentAccount": "30101810400000000225",
  "address": "г. Москва...",
  "legalAddress": "г. Москва...",
  "contractor": "ContractorCompany|ContractorHuman"
}
```

## OfferRow (позиция в счёте/сделке)

```json
{
  "contentType": "OfferRow",
  "offer": {"contentType": "Offer", "id": "string"},
  "name": "Услуга разработки",
  "count": 2,
  "price": {"contentType": "Money", "value": 1000},
  "discount": {
    "contentType": "Discount",
    "type": "percent",
    "valueInMain": 10
  },
  "tax": {"contentType": "Tax", "id": "string"}
}
```
