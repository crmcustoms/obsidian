# Задачи (Task / Todo) — Megaplan API v3

← [[Megaplan API — Index]]

Два типа: `Task` (проектные задачи) и `Todo` (дела/события/встречи).

---

## Эндпоинты — Task

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/task` | Список задач |
| GET | `/api/v3/task/{id}` | Получить задачу |
| POST | `/api/v3/task` | Создать задачу |
| POST | `/api/v3/task/{id}` | Обновить задачу |
| DELETE | `/api/v3/task/{id}` | Удалить задачу |
| POST | `/api/v3/task/{id}/doAction` | Действие (смена статуса) |
| GET | `/api/v3/task/{id}/subTasks` | Подзадачи |
| GET | `/api/v3/task/{id}/deals` | Связанные сделки |
| GET | `/api/v3/task/{id}/executors` | Исполнители |
| GET | `/api/v3/task/{id}/auditors` | Аудиторы |
| GET | `/api/v3/task/extraFields` | Доп. поля задач |
| GET | `/api/v3/project` | Список проектов |
| POST | `/api/v3/project` | Создать проект |
| GET | `/api/v3/project/{id}` | Получить проект |

## Эндпоинты — Todo (Дела)

| Метод | URL | Описание |
|-------|-----|---------|
| GET | `/api/v3/todo` | Список дел |
| GET | `/api/v3/todo/{id}` | Получить дело |
| POST | `/api/v3/todo` | Создать дело |
| POST | `/api/v3/todo/{id}` | Обновить дело |
| DELETE | `/api/v3/todo/{id}` | Удалить дело |
| POST | `/api/v3/todo/{id}/doAction` | Выполнить действие |
| POST | `/api/v3/todo/{id}/{changeMode}` | Сменить статус |
| GET | `/api/v3/todo/{entityId}/deals` | Сделки дела |
| GET | `/api/v3/todo/{entityId}/comments` | Комментарии |
| POST | `/api/v3/todo/{entityId}/comments` | Добавить комментарий |

| GET | `/api/v3/todoStatus` | Статусы дел |

## Статусы Task

| Значение | Смысл |
|----------|-------|
| `created` | Создана |
| `assigned` | Назначена |
| `accepted` | Принята |
| `done` | Выполнена |
| `completed` | Завершена |
| `rejected` | Отклонена |
| `cancelled` | Отменена |
| `expired` | Просрочена |
| `delayed` | Отложена |
| `overdue` | Просрочена |
| `template` | Шаблон |

## Создание задачи

```json
POST /api/v3/task
{
  "contentType": "Task",
  "name": "Название задачи",
  "isUrgent": false,
  "isTemplate": false,
  "subject": "Описание задачи",
  "parent": {"contentType": "Project", "id": "1"}
}
```
