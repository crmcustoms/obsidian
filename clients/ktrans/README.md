# KTRANS — Автоматизація обліку оренди авто

## Інформація про клієнта

- **Назва:** KTRANS
- **Сфера:** Оренда автомобілів
- **Філіали:** 
  - Київ (лівий берег)
  - Київ (правий берег)
  - Київ (комерційний транспорт)
  - Вінниця
  - Львів
  - Хмельницький
- **Статус:** Довгострокове співробітництво
- **Поточний стан:** Планфікс впроваджено, базова структура працює

## Поточна структура Планфікс

- Автопарк = задачі-автомобілі з прив'язкою до філіалів
- Картки оренди = підзадачі до автомобілів
- Фіксується: термін оренди, сума, валюта, застава
- Статус авто змінюється на «В оренді» при активній оренді

## Технічні деталі

### Planfix API endpoints
- `POST /task` — створення задачі
- `GET /task/{id}` — отримання даних
- `POST /task/{id}` — оновлення
- `POST /file/upload` — завантаження файлів
- `GET /directory/{id}` — робота зі справочниками
- `POST /analytics` — формування звітів

### n8n workflow
- Webhook Telegram → обробка → Planfix API
- Експорт Planfix → Google Sheets (по розкладу)

### Особливості
- `silent=true` для масових операцій
- Ліміт: 100 результатів на запит
- Пагінація для великих обсягів
- UTC в API для часових зон
- Валідація полів перед відправкою
