## Аналітика TAX

| Параметр | Значення |
|----------|----------|
| Datatag ID | `9611` |
| Поле Tax Type | `58393` (тип: Справочник 17495) |

**Значення Tax Type (Directory 17495):**

| key | Ставка |
|-----|--------|
| 1 | 16% |
| 2 | 12% |
| 3 | 0% |

### Як визначити ставку з Мегаплану
```python
# taxTotal = sum * rate / (100 + rate)
# → rate = (taxTotal / (sum - taxTotal)) * 100
if tax_total == 0:
    rate = 0
else:
    rate = round((tax_total / (total - tax_total)) * 100)
# 16% → key=1, 12% → key=2, 0% → key=3
```

## API-патерни Планфікс

### Список задач по шаблону
```python
POST /rest/task/list
{
  "offset": 0,
  "pageSize": 100,
  "filters": [{"type": 325, "operator": "equal", "value": 15}],
  "fields": "customField132121,customField130207,customField138909"
}
```

### Оновити кастомне поле задачі
```python
POST /rest/task/{id}
{
  "customFields": [
    {"field": {"id": 138909}, "value": "15-10-2025"}
  ]
}
```

### Створити запис datatag (аналітика TAX)
```python
POST /rest/task/{id}/datatags/
{
  "dataTag": {"id": 9611},
  "items": [{
    "customFieldData": [
      {"field": {"id": 58393}, "value": 2}
    ]
  }]
}
```

### Отримати всі записи datatag TAX
```python
POST /rest/datatag/9611/entry/list
{
  "offset": 0,
  "pageSize": 1000,
  "fields": "key,task"
}
```

### Видалити запис datatag
```python
DELETE /rest/datatag/9611/entry/{key}
```
