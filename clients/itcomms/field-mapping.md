## Що потрібно для завершення маппінгу

### Від Планфікс
Виконати для кожного шаблону (15, 7691, 11):
```bash
# Взяти будь-яку задачу з шаблону та запросити всі поля
curl -X POST "https://itcomms.planfix.com/rest/task/list" \
  -H "Authorization: Bearer 6ca06006655c6e695c495a4705609c85" \
  -H "Content-Type: application/json" \
  -d '{"offset":0,"pageSize":1,"filters":[{"type":325,"operator":"equal","value":15}],"fields":"customFieldData"}'
```

### Від Мегаплану
Виконати з сервера Hetzner для кожного типу угоди:
```bash
# Розхідна угода (нал) — програма у Мегаплані відповідає шаблону 15
docker exec itcomms-scripts curl -s \
  "https://likhtman.megaplan.ru/api/v3/deal/{id}" \
  -H "Authorization: Bearer NzZkODN..." | python -m json.tool
```

## Відомий маппінг (поточний)

### Invoice (шаблон 21) ↔ Мегаплан рахунок

| Планфікс поле | ID | Мегаплан поле | Примітка |
|---------------|-----|---------------|---------|
| Invoice Number | 128167 | `number` | Текст, напряму |
| Invoice Payment Date | 128157 | `actualPaymentDate` | Дата |
| Payment Amount | 128161 | `sum.value` | Число |
| TAX (16%) | 9611/58393=1 | `taxTotal` / `sum` | rate ≥ 15% |
| TAX (12%) | 9611/58393=2 | `taxTotal` / `sum` | rate ≥ 10% |
| TAX (0%) | 9611/58393=3 | `taxTotal` = 0 | |
| ID сделки | 132121 | `id` | Megaplan deal ID |
