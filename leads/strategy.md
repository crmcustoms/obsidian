# Стратегия поиска клиентов на ТП

## Цель
Найти 3–5 клиентов на постоянную техническую поддержку (10–20К/мес каждый)

## Модель
- Пакеты: БАЗОВИЙ 10К, СТАНДАРТ 15К, ПОВНИЙ 20К
- Формат: фиксированная оплата, не погодинка
- Цель: 45–100К/мес стабильно

## Приоритет 1: Тёплая база (неделя 1–2)

### Кого писать
Все 48+ клиентов из базы, которым уже делал проекты

### Шаблон сообщения:
```
Привіт! 👋

Зараз я беру 1–2 компанії на постійну технічну підтримку. 
Формат: фіксована оплата щомісяця, всі технічні задачі 
(CRM, боти, сайт, автоматизація) — закриті.

Якщо у вас є задачі, які накопичуються або потребують 
постійної уваги — давайте обговоримо формат. 
Можна короткий дзвінок цього тижня?

---
Кращі кейси та деталі: https://crmcustoms.com/
```

### KPI: 20 сообщений → 5 ответов → 2 звонка → 1 клиент

## Приоритет 2: LinkedIn-аутрич (ежедневно)

### Фильтр поиска
- Страна: Украина
- Должность: Власник, CEO, Генеральний директор, Founder
- Компания: Manufacturing, Services, Б2Б
- Размер: 20–200 сотрудников
- Активность: посты за последний месяц

### Шаблон сообщения #1 (по боли):
```
Привіт, [Ім'я]!

Бачу що [Компанія] активно росте — круто. 

Зазвичай на цьому етапі з'являється типова проблема: 
власник тушить пожежі замість стратегії, бо немає 
людини яка закриває технічні питання.

Я працюю як технічний партнер для таких компаній — 
CRM, автоматизація, боти. Можу за 15 хвилин показати 
де у вас зараз витікають гроші та час.

Давайте короткий дзвінок цього тижня?
```

### Шаблон сообщения #2 (по результату):
```
Привіт, [Ім'я]!

Впровадив 48+ CRM-проєктів для виробничих та сервісних 
компаній. Зараз беру 2–3 компанії на постійну техпідтримку.

Формат: фіксована ціна, всі технічні задачі закриті, 
власник нарешті займається бізнесом а не рутиною.

Якщо цікаво — розкажу детальніше за 15 хвилин?
```

### Шаблон сообщения #3 (короткий):
```
Привіт!

Шукаю 2–3 компанії для постійної технічної підтримки. 
CRM, боти, автоматизація — фіксована ціна, без погодинки.

Можна обговорити за 15 хв?
```

### KPI: 10–15 сообщений/день → 2–3 ответа/день → 1 звонок/день → 1 клиент/неделя

## Приоритет 3: Telegram-сообщества

### Каналы для мониторинга
- «Бізнес в Україні»
- Галузевые чаты производителей
- EO Ukraine (если есть доступ)
- YPO Ukraine (если есть доступ)
- Чаты про CRM, автоматизацию

### Стратегия
Не продавать напрямую. Быть полезным:
- Отвечать на вопросы
- Делать мини-кейсы в комментариях
- Периодически постить про свои результаты

### Шаблон поста (раз в неделю):
```
Кейс тижня 🚀

Компанія: [ніша], 30 осіб
Проблема: CRM стоїть, ніхто не підтримує, 
власник тушить пожежі

Рішення: підтримка 15К/міс, 25 годин роботи
Результат: власник вперше за рік побачив 
звіти без свого участі

Шукаю ще 2 такі компанії. 
Якщо знайома ситуація — пишіть.
```

## Приоритет 5: Реклама (позже)

### Почему не сейчас:
- Cold traffic не конвертирует для этой услуги
- Нужен прогрев и доверие
- Сначала добываем клиентов руками, понимаем что работает
- Потом масштабируем через рекламу

### Когда включать:
- Есть 2–3 клиента
- Понятно какая аудитория конвертирует
- Есть бюджет 15–20К на тесты
- Готовы 2–3 месяца на настройку

## Скрипт первого звонка (диагностика)

### 1. Приветствие (2 мин):
```
Привіт! Дякую що знайшли час. 
Коротко про формат: 15 хв діагностики, безкоштовно, 
без зобов'язань. Розкажете про свою ситуацію?
```

### 2. Вопросы (10 мин):
- Скільки людей у компанії?
- Які системи зараз використовуєте? (CRM, боти, сайт)
- Хто зараз закриває технічні задачі?
- Скільки часу власник витрачає на рутину?
- Які задачі накопичилися і чекають?
- Який бюджет виділяли б на техпідтримку?

### 3. Рекомендация (3 мин):
```
Зрозуміло. На основі того що сказали — 
вам підходить пакет [БАЗОВИЙ/СТАНДАРТ/ПОВНИЙ].

Ось що конкретно входить: [перелік].
Ціна: [сума] на місяць.

Якщо цікаво — надішлю договір на ознайомлення. 
Можна подумати та обговорити деталі.
```
